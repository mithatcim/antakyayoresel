# LÜLÜ'NÜN MUTFAĞI

LÜLÜ'NÜN MUTFAĞI, Hatay / Antakya yöresel yemeklerini tanıtan modern ve responsive bir restoran tanıtım sitesidir. Projede taş fırın lezzetleri, mezeler ve geleneksel Hatay mutfağı sıcak bir görsel dil ile sunulur.

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript

## Sayfalar

- **Anasayfa:** Hero alanı, öne çıkan lezzetler ve yönlendirme bölümleri.
- **Lezzetler:** Yöresel yemek kartları ve görsel galeri modalı.
- **İletişim:** İletişim formu, form doğrulama ve işletme bilgileri.

Bu proje özel build adımı gerektirmeyen statik bir HTML, CSS ve JavaScript projesidir. Vercel üzerinde yayınlarken framework seçmeden statik proje olarak içe aktarılabilir. Giriş dosyası `index.html` dosyasıdır.
